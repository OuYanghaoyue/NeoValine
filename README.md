# NeoValine

一个基于 [Valine](https://github.com/xCss/Valine) 魔改的有人工审核再显示功能的评论系统。

---

由于中国大陆网站备案与管理的需求，对 Valine 进行魔改以加入评论先审核后显示的功能。

该项目有配套的后端程序：[https://github.com/UdUklm/NeoValineBackend](https://github.com/UdUklm/NeoValineBackend)
