# NeoValine

一个在 Valine 基础上修改的简单评论系统。

A simple comment system modified on Valine. 

---

由于中国大陆网站备案与管理的需求，对 Valine 进行魔改以加入评论先审核后显示的功能。

有为该项目配套后端程序的打算，而非使用第三方数据存储服务，**可能需要将后端代码部署在服务器上配合使用**。
